# Longest Palindromic Subsequence | Наибольшая палиндромная подпоследовательность

## English

Educational project written in C that finds the Longest Palindromic Subsequence (LPS) in a given string.

The solution uses dynamic programming and memoization to efficiently determine the longest palindrome that can be obtained from the original sequence without changing the order of characters.

### Features

* Longest Palindromic Subsequence search
* Dynamic programming approach
* Memoization optimization
* Case-insensitive character comparison
* Input validation
* Reconstruction of the optimal subsequence
* Display of removed characters and their positions

### Technologies

* C
* Dynamic Programming
* Memoization
* String Processing
* Algorithms and Data Structures

## Русский

Учебный проект на языке C, реализующий поиск наибольшей палиндромной подпоследовательности (Longest Palindromic Subsequence).

Решение использует динамическое программирование и мемоизацию для эффективного нахождения максимальной палиндромной подпоследовательности без изменения порядка символов.

### Возможности

* Поиск наибольшей палиндромной подпоследовательности
* Использование динамического программирования
* Оптимизация с помощью мемоизации
* Сравнение символов без учёта регистра
* Проверка корректности входных данных
* Восстановление найденной подпоследовательности
* Вывод удалённых символов и их позиций

### Технологии

* C
* Динамическое программирование
* Мемоизация
* Обработка строк
* Алгоритмы и структуры данных
